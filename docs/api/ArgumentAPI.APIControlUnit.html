ParseFile(string)

Reads in the options that are specified in a configuration file, and stores them in the APIControlUnit instance. This file will be in a CSV format, with parameters being enclosed in {}. Since defining arguments in both a file and code is possible sequentially, if there is a problem with the file, the user will be issued a warning, but the program will continue.
